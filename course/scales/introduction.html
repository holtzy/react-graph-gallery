Introduction to scales

Building a graph requires to transform a dimension (a numeric variable in your dataset) in a position in pixels. This is done using a fundamental dataviz concept called scale.

Before implementing it in the next lesson, let's describe a bit what this crucial concept of dataviz is.

How it actually works

The obvious part:

‚Üí For a value of 0, the circle should be placed at the extreme left of the SVG. This corresponds to cx = 0px.

‚Üí For the highest value in the dataset, 100, the circle should be positioned at the extreme right of the SVG. This corresponds to cx = width (i.e., 500px).

‚Üí For a value of 50, which is the midpoint of our dataset, the circle should be positioned at the center of the SVG. This corresponds to cx = width / 2 (i.e., 250px).

The math part:

For a value of 82, which is not an exact midpoint, you need to calculate the position proportionally.

The position can be calculated as:

// Linear scale equation
cx = (value / maxValue) * width

// cx = (82 / 100) * 500
// cx = 410px
